Bad Suika Game - Installation Instructions

WIN - use the badsuikagameinstaller.exe file to install
MAC - use the Bad Suika Game.pkg file to install

Installer for mac made with the help of Krish Chauhan

Controls:
- Left/Right Arrow Keys: Move fruit
- Spacebar: Drop fruit
- Number Keys (1-5): Navigate menus
- Click Music button for volume control

The game saves your scores to Documents\BadSuikaGame\leaderboard.csv

Music sourced from Musecore from users ccruz21 and Scarnaval
